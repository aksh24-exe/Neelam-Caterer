# Neelam Caterer Website

A modern catering website for **Neelam Caterer** built with React, Tailwind CSS, and shadcn/ui components — featuring a Rajasthani-inspired design.

## Project Structure

```
neelam-caterer/
├── frontend/     # React + Vite + Tailwind + shadcn/ui
└── backend/      # Reserved for future API (not implemented yet)
```

## Frontend

### Tech Stack

- **React 19** + TypeScript
- **Vite** for build tooling
- **Tailwind CSS v4** for styling
- **shadcn/ui** components (Button, Card, Dialog, Accordion, etc.)
- **React Router** for navigation
- **Lucide React** for icons

### Pages

- `/` — Home (Hero, About, Team, Services, Menu preview, Gallery, Testimonials, Contact)
- `/menu` — Full menu with cuisines (Gujarati, Rajasthani, Punjabi, Bengali, South Indian, Chaat)
- `/quote` — Request a quote form
- `/gallery` — Full photo gallery with lightbox

### Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
cd frontend
npm run build
npm run preview
```

# Neelam Caterer — Backend

This folder is reserved for the backend API. It is not implemented yet.

## Planned Features

- Contact form submission API
- Quote request handling
- Menu management
- Gallery upload & management
- Admin dashboard

## Suggested Stack

- Node.js + Express or Fastify
- PostgreSQL or MongoDB
- Email notifications (Nodemailer / SendGrid)

## Getting Started (Future)

```bash
cd backend
npm install
npm run dev
```

The frontend currently uses local form state — forms will connect to this API once implemented.
